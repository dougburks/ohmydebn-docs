The OhMyDebn menu allows you to launch apps, learn about OhMyDebn, and configure and control the system. You can activate this menu from the [Cinnamon menu](cinnamon-menu.md) or by pressing `Super + Alt + Space`.

![OhMyDebn Menu](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-menu.png)

The first option launches the separate [Apps menu](apps-menu.md).
