OhMyDebn includes [ufw](https://help.ubuntu.com/community/UFW) and configures it to deny inbound traffic.

You can manage firewall rules from a [terminal](terminal.md) using `sudo ufw`. If you prefer a graphical interface, you can launch [gufw](https://help.ubuntu.com/community/Gufw) via [Cinnamon menu](cinnamon-menu.md) or [Apps menu](apps-menu).
