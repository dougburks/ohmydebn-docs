OhMyDebn includes [ufw](https://help.ubuntu.com/community/UFW) and configures it to deny inbound traffic.

You can manage firewall rules from a [terminal](terminal.md) using `ufw` or from a graphical interface using [gufw](https://help.ubuntu.com/community/Gufw):
