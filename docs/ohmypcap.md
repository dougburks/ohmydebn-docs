If you're in cybersecurity or networking, then you may need to analyze pcap files and we've got just the tool for that: OhMyPCAP!

To launch OhMyPCAP just press `Ctrl + Alt + P`. On first launch, it will automatically install Docker and then pull the Docker image. Once the Docker image starts just click the hyperlink at the bottom of the terminal window to open the OhMyPCAP web interface in your browser.

You can read more about OhMyPCAP at https://github.com/dougburks/ohmypcap.
