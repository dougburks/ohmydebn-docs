## LibreOffice

[LibreOffice](https://www.libreoffice.org/) is installed to handle common file types like documents and spreadsheets.

## Xournal++

PDF files open by default in the [chromium web browser](browser.md). If you need to annotate PDFs, then you can use the included [Xournal++](https://xournalpp.github.io/).

## Ristretto

[Ristretto](https://docs.xfce.org/apps/ristretto/start) is the default image viewer.

## GIMP

[GIMP](https://www.gimp.org/) is the default image editor.
