## Virtualization

If you'd like to run a virtual machine (VM) on your OhMyDebn installation, you can install Virtual Machine Manager by running `ohmdyebn-virtualization-install`. Alternatively, you can launch the OhMyDebn menu and then select `Install`:

![OhMyDebn Virtualization Install](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-0-install.png)

then select `Utility`:

![OhMyDebn Virtualization Utility](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-1-utility.png)

then select `Virtualization`:

![OhMyDebn Virtualization Virtualization](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-2-virtualization.png)

then press Enter to continue:

![OhMyDebn Virtualization Prompt](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-3-prompt.png)

Installation complete:

![OhMyDebn Virtualization Installed](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-4-installed.png)

You can then start Virtual Machine Manager from the menu or by pressing `Ctrl + Super + V`.

![OhMyDebn Virtualization Machine Manager](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-5-vmm.png)

Once you've created a VM, if you want to create a snapshot then we recommend using the `internal` option:

![OhMyDebn Virtualization Snapshots](https://raw.githubusercontent.com/dougburks/ohmydebn-docs/refs/heads/main/images/ohmydebn-virtualization-6-snapshots.png)
